# opencv-safety-line-drawing

#### this file will help you to draw the reference line for Computer vison project usecase .

- required tools :
    - python== 3.6.9 
    - Opencv 
    
#### where to use :
- for example , suppose if you're tracking one object & you want to raise an alarm as object cross specific line ( then ypu'll use this file draw this critical line ) 
- as the Orientation / rotation  of camera changes , your critical line need to be change .
